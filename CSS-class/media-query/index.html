Media Queries

Media queries are a powerful feature of CSS that allow you to apply styles based on the characteristics of the device or viewport. They enable responsive design, ensuring that your website looks good on all devices.

In this example, we have a section with a background color that changes based on the width of the viewport. The colors change at specific breakpoints, demonstrating how media queries can be used to create a responsive layout.
