What is HTML?

HTML is a markup language that is used to create web pages. It is a standard markup language for documents designed to be displayed in a web browser.

What is CSS?

CSS is a style sheet language that is used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, or in other media.
